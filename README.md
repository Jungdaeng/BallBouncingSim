# Ball Bouncing Simulator

회전하는 12각형(도데카곤) 내부에서 공이 중력과 충돌에 따라 움직이는 웹 시뮬레이터입니다.

## 주요 기능

- 일정한 각속도로 회전하는 12각형
- **Add Ball** 버튼으로 12각형 내부에 공 추가
- 중력 적용
- 공과 벽의 충돌 처리
- 공과 공 사이의 충돌 처리
- 검은색 기반의 직관적인 UI

## 실행 방법

별도 설치 없이 브라우저에서 바로 실행할 수 있습니다.

1. 프로젝트 폴더에서 아래 명령어 실행

```bash
python3 -m http.server 8000
```

2. 브라우저에서 접속

```text
http://localhost:8000
```

## 사용법

1. 페이지가 열리면 기본 공 6개가 생성됩니다.
2. **Add Ball** 버튼을 누르면 공이 하나씩 추가됩니다.
3. **Reset** 버튼을 누르면 모든 공이 제거되고 다시 시작할 수 있습니다.
4. 공은 중력의 영향을 받아 아래로 떨어지고, 회전하는 12각형 벽 및 다른 공과 자연스럽게 충돌합니다.
